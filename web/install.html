Before You Flash

  1. Connect the ESP board to this computer over USB.
  2. Use Chrome, Edge, or another browser with Web Serial support.
  3. If the board does not enter flashing mode automatically, hold BOOT while connecting or resetting it.
  4. Press Install, select the serial port, and follow the ESP Web Tools prompts.
