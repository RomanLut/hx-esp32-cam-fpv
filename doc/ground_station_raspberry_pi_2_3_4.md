# Raspberry Pi 2/3/4 Ground Station

Preparing SD Card for **Raspberry PI GS**: [/doc/software_for_rpi.md](/doc/software_for_rpi.md)

For a desktop-style Raspberry Pi setup, the Ground Station software can also be built and run on Ubuntu: [/doc/running_gs_on_ubuntu.md](/doc/running_gs_on_ubuntu.md)

Use a Wi-Fi adapter that supports monitor mode and packet injection. **rtl8812au** is recommended; **AR9271** is also supported.

Raspberry Pi 4 is recommended for development over Raspberry Pi 2/3 due to higher CPU performance.

***Note that Raspberry Pi GS is not actively developed and tested. It might be dropped in future releases.***
