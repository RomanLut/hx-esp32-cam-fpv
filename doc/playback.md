# Playback

## Introduction

Playback lets the GS open recorded video files from local GS storage and replay them through the normal video display path.


## Keys

| Key | Function |
|---|---|
| `Up` / `Down` | Stop playback and return to the Playback menu. |
| `Enter` | Stop playback and return to the Playback menu. |
| `Left` | Decrease playback speed. |
| `Right` | Increase playback speed. |
| `Space` | Pause/resume. When paused, step forward one frame. |
| `AIR REC` / `R` | Stop playback and open delete confirmation for the selected/playing file. |
| `GS REC` / `G` | No action in the Playback menu or during playback. |
