# Eachine Sphere Link Ground Station

The Eachine Sphere Link package includes an **RTL8812AU** USB Wi-Fi adapter. That adapter can be used by Android, Oculus Quest, Ubuntu, Radxa, or Raspberry Pi Ground Station builds.

Use the guide for the platform where the adapter will be connected:

* [Android Ground Station](/doc/ground_station_android.md)
* [Oculus Quest Ground Station](/doc/ground_station_oculus_quest.md)
* [Ubuntu Ground Station](/doc/ground_station_ubuntu.md)
* [Radxa Zero 3W/3E Ground Station](/doc/ground_station_radxa_zero_3.md)
* [Raspberry Pi Ground Station](/doc/ground_station_raspberry_pi_zero_2w.md)

The **RTL8812AU** adapter supports **2.4GHz** and **5.8GHz** operation when used with compatible Ground Station software and regional channel configuration.
