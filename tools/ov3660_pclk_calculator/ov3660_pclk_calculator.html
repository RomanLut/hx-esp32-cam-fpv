Frequency Chain

Notes:

  • Clock formulas (from ov3660.c calc_sysclk, PLLS registers 0x303A-0x303D):
    REFIN   = XVCLK / pll_pre_div;                    // /1, /1.5, /2, /3      (0x303D[5:4])
VCO     = REFIN * pll_multiplier * pll_root_2x;   // multiplier 1..31      (0x303B)
PLLCLK  = pll_bypass ? XVCLK : (VCO * 2 / pll_sys_div / seld5_2x);  // seld5: /1, /2, /2.5
SYSCLK  = PLLCLK / 4;
PCLK    = PLLCLK / 2 / (pclk_manual ? pclk_div : 1);
  • FPS formula: FPS = SYSCLK / (HTS × VTS). In binning mode the driver programs VTS = total_y/2 + 1.
  • VCO range is not published for the OV3660 PLLS; stock configs run 160-200 MHz, project presets up to ~293 MHz. Values above 400 MHz are flagged.
  • High FPS presets (SYSCLK ~72 MHz) need hardware validation — stock configs never exceed 50 MHz SYSCLK.
  • 1024x576 / 1280x720 cannot reach 30 FPS: they exceed the max binned 16:9 size (960x540), so the sensor scans the full 1080-line window (~19 FPS at 55 MHz SYSCLK). Tightened blanking at 72 MHz was tried and does not work on hardware.
  • All frequencies in Hz
